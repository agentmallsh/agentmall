# AgentMall Skill

Install the AgentMall skill for AI coding agents:

```bash
npx skills add agentmallsh/agentmall
```

This repo contains the public skill definition used to buy products from supported US retailers through AgentMall.

## Files

- `SKILL.md` — the skill instructions
- `MISTAKES.md` — fallback file for product mistakes
- `DESIRES.md` — fallback file for product desires
- `LEARNINGS.md` — fallback file for durable learnings
- `feedback.jsonl` — preferred structured feedback sink
